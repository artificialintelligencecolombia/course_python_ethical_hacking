==============================
IPCONFIG
==============================

Provides detailed info about the network interfaces on a Windows machine.

---------------------
Ethernet Adapter (PC Nic)
---------------------

- Represents a physical or virtual network interface card (NIC) used to connect the system to a LAN or the internet.
- Each interface has a unique MAC address and is assigned an IP address.

Ethernet Adapter Details
---------------

- **DNS Suffix**: Specifies the DNS domain name associated with this interface. In this example: `.home`.
- **IPv6 Address**:
	- **Link-local Address**: `fe80::9d36:ba56:835f:107c%17` - Automatically configured local address for IPv6.
- **IPv4 Address**: 
	- **Address**: `192.168.1.14` - The IPv4 address assigned to the interface.
	- **Subnet Mask**: `255.255.255.0` - Determines the network portion of the IPv4 address.
	- **Default Gateway**: `192.168.1.254` - Used to access external networks (like the internet).

### Adaptador de Ethernet VirtualBox Host-Only Network:

- **IPv6 Address**:
	- **Link-local Address**: `fe80::959d:4a1:d666:345c%9`.
- **IPv4 Address**:
	- **Address**: `192.168.56.1`.
	- **Subnet Mask**: `255.255.255.0`.
	- **Default Gateway**: Not configured - This interface is used for host-only communications within VirtualBox.

### Adaptador de Ethernet VMware Network Adapter VMnet1:

- **IPv6 Address**:
	- **Link-local Address**: `fe80::111a:f37:405c:c93f%11`.
- **IPv4 Address**:
	- **Address**: `192.168.224.1`.
	- **Subnet Mask**: `255.255.255.0`.
	- **Default Gateway**: Not configured - Used for isolated network configurations in VMware.

### Adaptador de Ethernet VMware Network Adapter VMnet8:

- **IPv6 Address**:
	- **Link-local Address**: `fe80::195b:4b4d:22b5:8de0%6`.
- **IPv4 Address**:
	- **Address**: `192.168.11.1`.
	- **Subnet Mask**: `255.255.255.0`.
	- **Default Gateway**: Not configured.
